Deploy

Create your own propshop

Good news, the code that runs this website is open-source. This means you can deploy a public or private propshop on a server of your choice.

Guide

  1. Install Ruby, nginx, and passenger

    There are numerous tutorials on other sites that describe the process setting up Ruby, nginx, and passenger.

  2. Clone the repository 
    hg clone https://bitbucket.org/osrf/thepropshop
  3. Setup thepropshop 

          cd thepropshop
      sudo bundle install
      RAILS_ENV=production rake db:seed 
      RAILS_ENV=production rake db:migrate
      RAILS_ENV=production rake assets:precompile

  4. Point nginx to thepropshop code. Below is an example for nginx. 
          server {
        listen       80;
        server_name  localhost;
        root /home/ubuntu/thepropshop/public;
        passenger_enabled on;

        location / {
          rack_env production;
          passenger_enabled on;
          root /home/ubuntu/thepropshop/public;
          try_files $uri $uri/ /index.html;
        }
      }
  5. Restart nginx: 
    sudo service nginx restart
  6. And you're done. Point a browswer to your propshop, and enjoy.
